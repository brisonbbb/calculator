# Red Scientific Calculator

A GUI scientific calculator built using Java Swing.

## Features
- Basic arithmetic operations (+, -, *, /)
- Scientific functions (sin, cos, tan)
- Square and square root
- Numerical differentiation
- Numerical integration
- Matrix addition
- Red themed user interface

## Language
Java

## How to Run
1. Compile:
   javac RedScientificCalculator.java

2. Run:
   java RedScientificCalculator
